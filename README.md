# MeowMusicSync

Бот для двусторонней фоновой синхронизации любимых треков между **Spotify** и **Яндекс Музыкой**.

## Как это работает
Скрипт раз в 10 минут опрашивает обе платформы и синхронизирует новые добавленные треки. 
Используется локальный кэш (манифест), поэтому опрашиваются только новые треки, что экономит трафик и не долбит API.
Для сопоставления названий используется умный алгоритм `fuzzywuzzy`, который умеет отбрасывать мусорные слова, но жестко штрафует, если на одной платформе это ремикс/кавер, а на другой оригинал.

## Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/meowflyx/MeowMusicSync.git
   cd MeowMusicSync
   ```

2. Установите зависимости:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

3. Переименуйте `.env.example` в `.env` и впишите свои токены:
   - `TG_BOT_TOKEN` и `TG_ADMIN_ID` — для телеграм-бота (чтобы получать уведомления об ошибках и управлять процессом).
   - `SPOTIPY_CLIENT_ID`, `SPOTIPY_CLIENT_SECRET`, `SPOTIPY_REDIRECT_URI` — ключи приложения из Spotify Developer Dashboard (редирект ставьте `http://127.0.0.1:8888/callback`).
   - `YANDEX_MUSIC_TOKEN` — токен от Яндекс Музыки (можно вытащить через расширения браузера).

4. **ВАЖНО**: Первая авторизация Spotify!
   Запустите `auth_spotify.py`:
   ```bash
   python3 auth_spotify.py
   ```
   Скрипт выдаст ссылку. Перейдите по ней в браузере, согласитесь с доступом, скопируйте URL, на который вас перекинуло (даже если там страница ошибки локалхоста), вставьте обратно в терминал и нажмите Enter. Сгенерируется файл кэша `.cache`.

5. Запуск бота:
   ```bash
   python3 main.py
   ```

## Команды бота
- `/sync_all` — Принудительно запустить полную двустороннюю синхронизацию.
- `/sync_ym_sp` — Синхронизация Яндекс -> Spotify.
- `/sync_sp_ym` — Синхронизация Spotify -> Яндекс.
